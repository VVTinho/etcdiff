 .TH man 8 24 Jan 2015 1.0 etcdiff man page
 .SH NAME etcdiff \- create a new LDAP user
 .SH SYNOPSIS etcdiff [USERNAME]
 .SH DESCRIPTION etcdiff is high level shell program for adding users to LDAP server. On Debian$
 .SH OPTIONS The etcdiff does not take any options. However, you can supply username.
 .SH SEE ALSO useradd(8), passwd(5), nuseradd.debian(8)
 .SH BUGS No known bugs.
